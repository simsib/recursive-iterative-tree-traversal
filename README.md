# recursive-iterative-tree-traversal

### for react app 
1. have webpack-dev-server instaled globaly
```npm install -g webpack-dev-server```
2. ```npm install```
3. ```npm run dev```
4. visit [localhost](http://localhost:8080)

---
### for run simple scripts
```cd "node scripts"```
for recursive run 
```node Recursive.js``` 
or for iterative run
```node Iterative.js```
